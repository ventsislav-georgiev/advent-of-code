# Advent Of Code
ref: https://adventofcode.com


## Prerequisites
`$SESSION_KEY` env var is expected. Available on the website in cookie named `session`.

## Usage
```sh
make go year=2022 day=01 task=2 
```
or
```sh
make rust year=2022 day=03 task=1 
```

## Benchmark (Golang only)
```sh
make bench day=10
```
